🤡 =USEAI() for Excel

Bring AI to any Excel cell, today. Free, open-source add-in — Bring Your Own API Key.

=USEAI("Summarize this data", A1:D50)

How It Works

Type =USEAI("Summarize this data", A1:D50) in any cell, and get instant AI-powered answers — no VBA, no macros, no server setup.

What Makes It Different

🔑

Bring Your Own Key (BYOK)

Works with Anthropic Claude, OpenAI GPT, and Google Gemini. You choose the model, you control the cost.

📄

File-Aware Prompts

Reference Word docs, PDFs, images, and URLs directly in your formulas with {C:\report.pdf} syntax. Content is extracted and sent automatically.

📈

Spills Like Native Excel

Returns tables, lists, or single values. Integrates naturally with your existing workflows.

⚡

Smart Caching

Identical prompts are served from memory, saving API calls and money.

🛡

Per-Provider Rate Limiting

Built-in safeguards so you never blow through your API budget.

📦

Zero Infrastructure

One .xll file, drag into Excel, done. No Python, no server, no cloud deployment.

Built for Professionals Who Live in Excel

Analysts, consultants, project managers, finance teams — anyone who needs AI assistance without leaving their spreadsheet.
